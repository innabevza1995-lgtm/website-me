## Сайт-лендинг — Бевза Инесса

Современный продающий лендинг + страница гайда.

### Что внутри
- **`index.html`**: основной лендинг
- **`guide.html`**: гайд (как прототип, но лучше)
- **`styles.css`**: стили
- **`script.js`**: меню, FAQ, копирование, подсветка оглавления, анимации, форма контакта
- **`assets/`**: изображения (включая `hero-photo.png`)

### Как запустить (самый простой способ)
Открой `index.html` двойным кликом в браузере.

### Как запустить “как на сервере” (рекомендуется)
В терминале, находясь в папке проекта:

```bash
python3 -m http.server 5173
```

Открой в браузере `http://localhost:5173`.

### Настройка контактов (Telegram / Email)
Открой `script.js` и в начале файла заполни:

```js
const SITE_CONFIG = {
  telegramUsername: "ваш_username_без_@",
  emailTo: "you@example.com",
};
```

После этого в форме появятся кнопки “Открыть Telegram / Открыть Email / Скопировать текст”.

### Деплой на GitHub Pages (по шагам)
1. Создай репозиторий на GitHub и загрузи файлы проекта.
2. В GitHub открой **Settings → Pages**.
3. Выбери **Deploy from a branch**, branch **main**, folder **/** (root).
4. Сохрани — через минуту появится ссылка.

Готово.
